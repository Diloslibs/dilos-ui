# Input Text Components

DilosUI comes with a input text component.


## Basic
### Standart
```html live
<div class="flex flex-col gap-4">
  <div class="grid grid-cols-3 gap-1">
    <div class="flex flex-col gap-1 p-2">
      <input type="text" class="d-input-text sm" placeholder="Placeholder"/>
    </div>
    <div class="flex flex-col gap-1 p-2">
      <input type="text" class="d-input-text md" placeholder="Placeholder"/>
    </div>
    <div class="flex flex-col gap-1 p-2">
      <input type="text" class="d-input-text md" placeholder="Placeholder" disabled/>
    </div>
  </div>
</div>
```
### Error
```html live
<div class="flex flex-col gap-4">
  <div class="grid grid-cols-2 gap-1">
    <div class="flex flex-col gap-1 p-2">
      <input type="text" class="d-input-text sm error" placeholder="Placeholder"/>
    </div>
    <div class="flex flex-col gap-1 p-2">
      <input type="text" class="d-input-text md error" placeholder="Placeholder"/>
    </div>
  </div>
</div>
```
