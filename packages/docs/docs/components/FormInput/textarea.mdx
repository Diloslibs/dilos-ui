# Textarea Components

DilosUI comes with a textarea component.

import Textarea from '@site/static/usage/v1/forminput/textarea/default/index.mdx';

<Textarea />


## Default

```html live
<div>
  <textarea class="d-textarea" placeholder="Placeholder"></textarea>
</div>
```

## Disable

```html live
<div>
  <textarea class="d-textarea" placeholder="Placeholder" disabled></textarea>
</div>
```

## Error

```html live
<div>
  <textarea class="d-textarea error" placeholder="Placeholder"></textarea>
  <span class="text-red-400 text-sm">Error message goes here</span>
</div>
```
