# Chip Components

DilosUI comes with a Chip component that can be used to trigger actions.

## Gray
### Default
``` html live
    <div class="flex items-center gap-5">
        <div class="d-chip xs gray"> 
            Label
        </div>

        <div class="d-chip md gray"> 
            Label
        </div>

        <div class="d-chip lg gray"> 
            Label
        </div>
    </div>
```

### Disabled
``` html live
    <div class="flex items-center gap-5">
        <div class="d-chip xs gray disabled"> 
            Label
        </div>

        <div class="d-chip md gray disabled"> 
            Label
        </div>

        <div class="d-chip lg gray disabled"> 
            Label
        </div>
    </div>
```

### Selected
``` html live
    <div class="flex items-center gap-5">
        <div class="d-chip xs gray selected"> 
            Label
        </div>

        <div class="d-chip md gray selected"> 
            Label
        </div>

        <div class="d-chip lg gray selected"> 
            Label
        </div>
    </div>
```
