# Button Components

DilosUI comes with a button component that can be used to trigger actions.


## Primary
### Default
```html live
<div class="flex flex-col gap-4">
  <div class="grid grid-cols-5 gap-1">
    <div class="flex flex-col gap-1 p-2">
      <div class="flex items-center justify-center col-span-1 py-4 flex-grow">
        <button class="btn btn-primary btn-xs">
          Button
        </button>
      </div>
    </div>
    <div class="flex flex-col gap-1 p-2">
      <div class="flex items-center justify-center col-span-1 py-4 flex-grow">
        <button class="btn btn-primary btn-sm">
          Button
        </button>
      </div>
    </div>
    <div class="flex flex-col gap-1 p-2">
      <div class="flex items-center justify-center col-span-1 py-4 flex-grow">
        <button class="btn btn-primary btn-md">
          Button
        </button>
      </div>
    </div>
    <div class="flex flex-col gap-1 p-2">
      <div class="flex items-center justify-center col-span-1 py-4 flex-grow">
        <button class="btn btn-primary btn-lg">
          Button
        </button>
      </div>
    </div>
    <div class="flex flex-col gap-1 p-2">
      <div class="flex items-center justify-center col-span-1 py-4 flex-grow">
        <button class="btn btn-disable btn-primary btn-lg">
          Button
        </button>
      </div>
    </div>
  </div>
</div>
```
