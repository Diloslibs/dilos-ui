# Tooltip Components

DilosUI comes with a tooltip component that can be used to trigger actions.

## Default
```html live
    <div class="flex flex-col gap-6">
        <div class="d-tooltip dark">
            Tooltip text here
        </div>

        <div class="d-tooltip dark">
            <div class="d-tip top top-left"></div>
            Tooltip text here
        </div>

        <div class="d-tooltip dark">
            <div class="d-tip top top-right"></div>
            Tooltip text here
        </div>

        <div class="d-tooltip dark">
            <div class="d-tip top top-center"></div>
            Tooltip text here
        </div>

        <div class="d-tooltip dark">
            <div class="d-tip bottom bottom-left"></div>
            Tooltip text here
        </div>

        <div class="d-tooltip dark">
            <div class="d-tip bottom bottom-right"></div>
            Tooltip text here
        </div>

        <div class="d-tooltip dark">
            <div class="d-tip bottom bottom-center"></div>
            Tooltip text here
        </div>

        <div class="d-tooltip dark">
            <div class="d-tip left"></div>
            Tooltip text here
        </div>

         <div class="d-tooltip dark">
            <div class="d-tip right"></div>
            Tooltip text here
        </div>
    </div>

    
```
